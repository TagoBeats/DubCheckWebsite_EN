# DubCheck — International Landing Page (audio-dubcheck.com)

English-language landing for international markets.

**Two personas:**
- `/narrators` — ACX / Audible audiobook narrators
- `/studios` — Localization studios / re-recording mixers (Netflix NOLS, Apple TV+, Disney+, Prime)

**Sister site (German, not touched here):** `audio-dubcheck.de` — DACH broadcast / advertising / sync. Lives in `New_DubCheck_Website/`.

## Stack
Next.js 15 (App Router) · React 19 · TypeScript · Tailwind 3.4 · Vercel

## Dev
```bash
npm install
npm run dev
```

## Deploy
Auto-deploys via Vercel on push. Domain `audio-dubcheck.com` connects in Vercel dashboard.

## Strategy context
Pre-sale-first: this site sells early access to a future standalone macOS/Windows app. V1 build only starts after first paid pre-sales.
